## 4.  프로젝트 실행 명령어 📱

## ****Getting start****

### Compile

```bash
./gradlew compileJava

java -cp build/classes/java/main ClassName
```

### Run

```bash
./gradlew run
```

### Run test

```bash
./gradlew test
```
